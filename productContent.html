海思Hi3559Av100开发板

产品概述

处理器内核:

1.双核 ARM Cortex A73@1.8GHz，32KB I-Cache， 64KB D-Cache /512KB L2 cache

2.双核 ARM Cortex A53@1.2GHz，32KB I-Cache， 32KB D-Cache /256KB L2 cache

3.单核 ARM Cortex A53@1.2GHz，32KB I-Cache， 32KB D-Cache /128KB L2 cache

4.支持 Neon 加速，集成 FPU 处理单元

GPU:

1. 双核 ARM Mali G71@900MHz，256KB cache

2. 支持 OpenCL 1.1/1.2/2.0

3. 支持 OpenGL ES 3.0/3.1/3.2

Sensor Hub:

1. 集成 ARM Cortex M7@192Mhz

2. 集成 PMC 电源控制单元，PMC 只支持外部复位

3. 支持内部 POR

4. 支持通用外设 IP（UART/SPI/I2C/PWM/GPIO/LSADC）

5. 支持 2 通道 LSADC，7 个 UART 接口，8 个 PWM 接口

NNIE:

1. 支持 Alexnet、VGG16、Googlenet、Resnet18、Resnet50等多种分类神经网络

2. 支持 Faster R-CNN、YOLO、SSD、RFCN 、YoloV2 等多种目标检测神经网络

3. 支持SegNet、FCN 等场景分割网络

4. 2.0Tops 神经网络运算性能

5. 支持完整的 API 和工具链（编译器、仿真器），易于适配客户定制网络

视频编码：

1. H.264 BP/MP/HP

2. H.265 Main Profile/Main 10 Profile

3. H.264/H.265支持 I/P/B 帧， 支持 MJPEG/JPEG Baseline编码

4. H.264 编码可支持最大分辨率为 8192 x 8640 Pixel

5. H.265 编码可支持最大分辨率为 16384 x 8640Pixel

6. H.264/H.265 多码流实时编码能力：- 7680 x4320@30fps+1080P@30fps+7680 x 4320@2fps抓拍

7. 支持最大JPEG抓拍性能7680 x 4320@15fps

8. 支持CBR/VBR/AVBR/FIXQP/QPMAP五种码率控制模式，输出码率最大值到200Mbps

9. 支持8个感兴趣区域（ROI）编码

10. 支持 H.264 BP/MP/HP

11. 支持 H.265 Main Profile/Main 10 Profile

12. 支持 JPEG/MJPEG 解码

13. 最高支持到H264/H.265 7680 x 4320@30fps 或H.264/H.265 3840 x2160@120fps

14. 最高支持到7680 x 4320@15fps JPEG解码

智能视频处理：

1. 提供视觉计算处理能力

2. 四核 DSP@700MHz，32K I-Cache /32K IRAM/512KB

DRAM:

1. 双核 NNIE@840MHz 神经网络加速引擎

2. 内置双目深度检测单元
